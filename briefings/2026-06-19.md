## ⚡ 本期重点推荐

### ① chopratejas/headroom — 把你的 token 消耗砍掉 60-95%，还能输出一样的结果

> 压缩工具输出、日志、文件、RAG 块之后再喂给 LLM。支持 library / proxy / MCP server 三种接入方式，直接挂到 Claude Code、Cursor 上就能用。38K ⭐ 说明大家真的肉疼 token 费。

**🔥 值得马上试**

· 仓库：https://github.com/chopratejas/headroom

**为什么推荐**：每个用 agent 的人都遇到的问题——工具输出太长，token 烧得飞快。headroom 不是简单截断，而是智能压缩，保证下游 LLM 理解质量的前提下减少 60-95% 的 token。MCP server 模式意味着可以零侵入挂到现有 agent 工具链上。配合这周 HN 热帖「Bad MCP design costs your agent 5x more tokens」一起看，上下文优化正在从 nice-to-have 变成 agent 工程的必选项。

### ② NVIDIA/SkillSpector — 大厂正式入场 agent skills 安全扫描

> NVIDIA 出品的 AI agent skills 安全扫描器，检测技能文件中的漏洞、恶意模式和安全隐患。Apache 2.0 开源。8K ⭐ 两周内飙升。

**⚡ 值得关注进度**

· 仓库：https://github.com/NVIDIA/SkillSpector

## 📊 趋势洞察

### 1. MCP 从协议变成基础设施——官方 SDK + 注册表 + 企业背书三管齐下

Google 和 Microsoft 分别合维护 MCP 的 Go/C# 官方 SDK，微软还发布了官方 MCP server catalog。加上社区注册表上线、Unreal Engine 加入，MCP 已经不再是 "Anthropic 的实验性协议" 而是获得大厂正式背书的基础设施层。这个势头下，未来 3-6 个月会出现大量 "X 软件 + MCP server" 的组合，agent 可用的工具种类将指数级增长。

传导链：官方 SDK → 降低开发门槛 → 更多 MCP server → 更多 agent 可调用工具 → agent 能力边界扩大。

### 2. 上下文成本焦虑催生新工具品类——token 压缩和上下文优化成为 agent 工程的核心关注点

headroom（38K⭐）和 context-mode（17K⭐）的高速增长不是偶然。当一个 Claude Code session 动辄消耗数百万 token、成本达到几十美元时，token 效率不再是可以延后优化的细节，而是决定 agent 能否规模化使用的生死线。HN 热帖 "Bad MCP design costs your agent 5x more tokens" 进一步放大了这个焦虑。预测：未来会出现更多专门优化 agent 上下文效率的工具，可能发展出类似 webperf 那样的独立子领域。

传导链：token 成本上升 → 工具侧优化需求 → headroom/context-mode 爆发 → MCP 设计最佳实践形成 → agent 工程从「能不能用」到「划不划算」。
