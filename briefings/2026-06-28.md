## ⚡ 本期重点推荐

### ① OpenKnowledge — 开源 AI-first 知识管理，替代 Obsidian/Notion
OpenKnowledge 是一个开源的知识管理平台，用 AI 优先的架构重新思考了笔记工具的形态。HN 374 分、171 评论的热度说明开发者社区对 Obsidian/Notion 的"AI 原生替代品"有真实需求。它不做传统的文件夹树，而是以知识图谱为核心组织信息结构，AI 自动提取实体、关系和摘要——这在 Agent Skills 时代尤其重要：你的笔记可以被编程化地消费。

**🔥 值得马上试** — 如果你已经在用 Obsidian/Notion 做知识管理，这是一个值得切换考察的选项。开源、AI-first 的架构意味着更低的迁移成本、更高的自动化可能性。

· GitHub: https://github.com/inkeep/open-knowledge
· 网站: https://openknowledge.ai

### ② headroom — Agent 上下文压缩层，Token 消耗直降 60-95%
headroom 不是这周才出现的——它已经连续三周占据 GitHub Trending——但本周它从 "trending tool" 正式跨越到 "生态基础设施" 阶段。53K+ 星、MCP server 模式、跨 agent 框架兼容性（Claude Code / Codex / Cursor / Aider），以及本周新增的跨 agent 内存共享实验性功能，让它成为 2026 年 agent 工程领域最重要的工具之一。

**🔥 值得马上试** — Token 成本是 agent 可用性的沉默杀手。headroom 用一行命令 (`headroom proxy --port 8787`) 即可接入现有 agent 工作流，无需改代码。如果你在重度使用 Claude Code 或 Codex，这是本季度最值得配的工具。

· GitHub: https://github.com/headroomlabs-ai/headroom
· 定位：库 + 代理 + MCP Server 三位一体

## 🎯 Skills Watch

本周 Agent Skills 生态的关键动态：

- 🔥 **Google DESIGN.md 规范发布** — Google Labs 推出面向 AI agent 的结构化设计系统描述格式。不是 SKILL.md 的竞品，而是互补——DESIGN.md 定义"系统长什么样"，SKILL.md 定义"你怎么操作它"。设计 token 变成 agent 可消费的数据。
- 🔥 **Anthropic 公开 Agent Skills 仓库** — anthropics/skills 上线，这是继微软 skills、Google skills、Vercel agent-skills 之后的第四个科技巨头官方 skills 仓库。Agent Skills 从"社区实践"向"企业基础设施"的转变已不可逆。
- ⚡ **SkillsBench 首次质量审计** — 对 47,150 个公开 skills 打分，平均质量仅 6.2/12。但经过筛选的 skills 将 agent 通过率平均提升 16.2 个百分点。这标志着 skills 生态从"野蛮生长"进入"质量治理"阶段。
- ⚡ **Snyk ToxicSkills 安全研究** — 扫描 22,511 个 skills 发现 140,963 个问题，36% 存在 prompt injection 风险。Agent Skills 的安全治理将成为 2026 下半年的核心议题。
- ⚡ **Skills 生态规模突破里程碑** — SkillsMP 索引约 190 万公开 skills（GitHub 全量扫描），Skiln 索引 16,000+ MCP server + skills 条目。Agent Skills 从"几百个"到"百万级"只用了一年。
- 🔥 **Microsoft Agent Framework .NET 1.8.0** — 新增 MCP 长时任务支持、Foundry Toolbox MCP Skills 发现、ClaimsIdentity agent session 隔离，以及与 Python 版 Handoff Orchestration 对等。
- ❄️ **Solo.io MCP Gateway** — 6月24日发布，专为 AI agent 开发设计的 MCP 安全网关，填补 MCP 运行时防护的空白。
- ❄️ **Salesforce AgentExchange GA** — 10,000+ 应用、2,600+ Slack app、1,000+ agent/MCP，企业级 agent marketplace 正式运营。

### 1. Agent 上下文优化从"优化"变成"必需品"

### 2. Agent Skills 从社区玩具到企业治理
四家科技巨头的官方 skills 仓库（MS/Google/Anthropic/Vercel）+ SkillsBench 质量审计 + Snyk 安全扫描 + 百万级 skills 目录——Agent Skills 生态的成熟曲线正在加速。下半年最值得关注的不是"又多了多少个 skills"，而是 skills 的安全治理、质量认证、企业级注册表管理这些"成人化"基础设施。

## 💎 本周最值得试的新技能

**给 Claude Code 用户：装上 headroom MCP server**

```bash
# 一行接入，无需改现有工作流
headroom proxy --port 8787
# 然后在 .mcp.json 中配置
```

理由：如果你每天用 Claude Code/Codex 超过 1 小时，headroom 可能在第一周就帮你的月 token 账单打对折。这不是夸张——实测 CodeCompressor 对 verbose log 和 JSON 输出的压缩率可达 90%+，且支持可逆解压（agent 需要时可请求原始数据）。
