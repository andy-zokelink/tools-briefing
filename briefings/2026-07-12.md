## ⚡ 本期重点推荐

### ① Claude Code Week 28 — 内置浏览器 + /doctor 自修复，从"编码 agent"到"开发者工作站 OS"

Claude Code 本周（Week 28, Jul 6-10）发布了可能是今年最重要的功能组合拳：

- **内置浏览器** — Claude Code Desktop 现在可以直接打开外部网站、文档、设计稿，像操作本地 dev server 预览一样浏览和交互。浏览器沙箱化且可配置：你可以选择是否持久化浏览会话，安全分类器会审核在外部网站上的操作。
- **/doctor（别名 /checkup）自修复** — 从只读诊断升级为自动驾驶模式。它检查安装健康状态、找出未使用的 skills/MCP servers/plugins vs 其上下文成本、去重本地 CLAUDE.md 与已提交版本的差异、建议修剪 Claude 可以从代码库自行推导的 CLAUDE.md 内容、标记慢 hooks。先报告发现，确认后再执行修改。
- **其他提升**：
  - 对上下文无法解析的变量禁止 `rm -rf`
  - `/cd` 现在支持目录补全
  - `commit-push-pr` 自动允许 push 到配置的 remote
  - `login` 支持 Anthropic 运营的公共网关端点
  - Agent/Code Review 在 Opus 4.8 上质量全面提升

🔥 **值得马上试** — `/doctor` 一键修复环境，内置浏览器让 Claude Code 真正成为"开发者工作站的完整 OS"而非只是"代码生成器"。

## 📊 趋势洞察

### 趋势 1：Agent Skills 从"小众工具"到"GitHub 第一品类"

本周 GitHub Trending 前 8 全部是 AI agent 项目，5 个是 skills 仓库。这不是某个特定 repo 的爆发，而是整个品类的集体崛起。

mattpocock/skills 的 164K★ 尤其说明了一个深层信号：开发者不再仅仅是"使用 AI 来写代码"，而是在"教 AI 如何更好地写代码"。Skills 正在成为 API / MCP 之后的第三大 AI 扩展机制——API 告诉 agent 能访问什么，MCP 告诉 agent 能操作什么，Skills 告诉 agent 该怎么做。

Google stitch-skills 和 AWS OpenSearch Skills 的加入意味着大型科技公司也开始认真押注这个方向。Skills 不是一时的热潮——它是 agent 使用方式的范式转移。

**传导链：** Skills 标准化 → 大型平台（Google/AWS/Anthropic）官方 Skills 矩阵成型 → Skills 分发市场爆发 → 开发者从"写代码"转向"写教 AI 写代码的代码"。

### 趋势 2：Claude Code 从"编码 agent"进化为"开发者工作站 OS"

Claude Code Week 28 的内置浏览器 + /doctor 自修复两个功能组合在一起，标志着 Claude Code 正在跨越"写代码"这个单一职能边界，开始接管整个开发者工作流：

- **浏览层**：内置浏览器让 Claude 能看文档/设计稿/外部网站
- **诊断层**：/doctor 自修复让 Claude 能诊断和优化自身环境
- **协作层**：子代理深度 5 层、Artifacts 共享、后台会话
- **自动化层**：commit-push-pr 自动链、Shell 自动应答

Claude Code 不再只是一个"代码生成器"。它是一个以终端/桌面为前端的、完整的开发者工作站操作系统。这与其他 AI IDE（Cursor、Windsurf、Antigravity）的路线差异正在拉大——Claude Code 选择"极简终端 + 极强 Agent"，而其他选择"完整 IDE + 辅助 Agent"。

## 💎 本周最值得试的新技能

**跑一次 `claude /doctor` 清理和优化你的 Claude Code 环境**

本周 Claude Code 的 `/doctor`（别名 `/checkup`）从只读诊断升级为自修复模式。运行一次，它会：
1. 检查安装健康状态
2. 找出未使用的 skills/MCP servers/plugins 及其上下文成本
3. 去重本地 CLAUDE.md 与已提交版本的差异
4. 建议修剪 Claude 可以从代码库自行推导的 CLAUDE.md 内容
5. 标记慢 hooks

然后问你要不要修复——确认后自动完成所有优化。这是一个零成本的环境体检，强烈推荐所有 Claude Code 用户试试。

**装一个 Agent Skills 试试**
选一个入门的：`mattpocock/skills`（TypeScript/Git 实用 skills）或 `addyosmani/agent-skills`（Web 性能工程 skills）。克隆到项目根目录的 `.claude/skills/` 下，Claude Code 自动发现和使用。
